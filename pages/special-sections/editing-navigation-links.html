Editing Navigation Links

2 min read

After adding a navigation link, you may need to update its name or change its destination. The website builder allows you to modify any existing navigation link easily.

Steps to Edit a Navigation Link:

  1. Open the Navigation Manager
    • Click Sections Manager from the left bar.
    • Hover over the navigation section and click Navigation Manager.
    Alternatively:
    • Click Theme Settings or Text Manager, hover over any link in the navigation bar, and click Change Nav Link.
  2. Locate the Link to Edit
    • In the popup that appears, you'll see a list of existing navigation links.
    • Find the link you want to modify.
  3. Edit the Link
    • Click on the plus (+) icon next to the navigation link.
    • Two fields will appear:
      • Menu Name: Update how the link appears in the navigation bar.
      • URL: Change the destination of the link.
    • Click Save to confirm the changes for this specific link.
  4. Reordering the Navigation Links (Optional)
    • Drag and drop the links to arrange them in your preferred order.
  5. Click Apply to save all modifications and update the navigation menu.
