Removing navigation links

1 min read

If you no longer need a navigation link, you can easily remove it from the Navigation Manager.

Steps to Remove a Navigation Link:

  1. Open the Navigation Manager by either:
    • Clicking on Sections Manager, hovering over the navigation section, and selecting Navigation Manager.
    • Clicking on Theme Settings or Text Manager, hovering over any navigation link, and selecting Change Nav Link.
  2. In the popup, locate the list of navigation links.
  3. Find the link you want to remove and click on the trash icon.
  4. Click Apply to save the changes and update the navigation menu.
