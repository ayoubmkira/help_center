Changing the Navigation Type

1 min read

You can modify the navigation type to adjust how it behaves on the page.

Steps to change the navigation type

  1. Click on Sections Manager in the left bar.
  2. Hover over the navigation section to reveal options.
  3. Click on the Change Section Settings button.
  4. In the settings popup, locate the Navigation Type input list.
  5. Choose the desired type from the available options (Default, Absolute, Fixed, etc.).
  6. Click Apply to save the changes.
