Edit HTML code

2 min read

The Edit HTML Code option allows you to make advanced customizations by directly editing the HTML of a section. This is ideal for users who want full control over the structure and content of the section.

Edit HTML code

  1. Click on AI Sections Manager.
  2. Hover over the section you want to adjust.
  3. Click on the Code Editor icon.
  4. Make your changes.
  5. click on Apply button.

Editing the HTML code gives you the flexibility to implement custom features, such as adding new elements, changing the structure, etc. This is recommended for users with coding experience who want to fine-tune their section's content.
