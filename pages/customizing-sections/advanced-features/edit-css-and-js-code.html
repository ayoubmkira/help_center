Edit CSS and JS Code

2 min read

The Edit CSS and JS Code option allows you to customize the appearance and functionality of a section. By modifying the CSS, you can adjust styles such as colors, fonts, spacing, and animations. Editing the JS code enables you to add custom interactions and dynamic behaviors to enhance your section’s functionality.

Edit CSS and JS Code

  1. Click on AI Sections Manager.
  2. Hover over the section you want to modify.
  3. Click on the Section Settings button.
  4. Locate the CSS & JS Editor options and make your changes.
  5. Click the Apply button to save and apply the modifications.

This feature provides flexibility for advanced customization, allowing you to fine-tune your section’s design and behavior to better fit your website’s needs.
