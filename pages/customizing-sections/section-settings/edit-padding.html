Edit Padding

3 min read

Padding allows you to adjust the spacing at the top and bottom of a section to control its layout and improve readability.

Edit Padding

  1. Click on AI Sections Manager icon.
  2. Hover over the section you want to adjust.
  3. Open the Section Settings panel.
  4. Locate the Padding option.
  5. Increase or decrease the top and bottom padding values as needed.
  6. Click on Apply button.

This ensures proper vertical spacing between sections for a balanced design.
