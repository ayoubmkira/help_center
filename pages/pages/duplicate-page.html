Duplicate page

1 min read

Duplicating a page allows you to create an exact copy of an existing page, saving time and effort when designing similar layouts. This feature is useful for maintaining a consistent design across multiple pages without starting from scratch.

How to duplicate a page

  1. Open the Pages Manager from the left panel.
  2. Locate the page you want to duplicate.
  3. Click the Duplicate button next to the page name.
  4. A new copy of the page will be created with the same content and settings.

Once duplicated, you can rename the new page and make modifications as needed without affecting the original.
