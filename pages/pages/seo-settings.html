SEO Settings

1 min read

Optimizing your page’s SEO settings helps improve its visibility in search engines and enhances user engagement. You can configure meta information and tracking options directly from the page settings.

Accessing and Editing SEO Settings

  1. Open the Pages Manager from the left panel.
  2. Locate the page you want to edit.
  3. Click the Page Settings button.
  4. In the settings popup, navigate to the SEO tab.
  5. Configure the following options:
    • Generate Meta With AI: Click this link to automatically generate SEO-friendly meta tags.
    • Meta SEO: Manually enter a custom meta description to improve search engine ranking.
    • Custom Pixel Integrations: Add JavaScript code to collect customer events for analytics and tracking.
  6. Click Apply to save your changes.

By optimizing SEO settings, you can enhance your website’s discoverability and improve user interaction.
