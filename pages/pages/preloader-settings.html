Preloader settings

1 min read

A preloader is a loading animation displayed before the page fully loads. This feature enhances user experience by providing visual feedback while content is being prepared.

Accessing and Editing the Preloader Settings

  1. Open the Pages Manager from the left panel.
  2. Locate the page you want to edit.
  3. Click the Page Settings button.
  4. In the settings popup, navigate to the Preloader tab.
  5. Choose one of the following preloader options:
    • None: Disable the preloader.
    • Image: Upload a custom image as a loading animation.
    • Icon: Select an icon to be displayed while the page loads.
  6. Click Apply to save your changes.

Using a preloader can enhance the perceived performance of your website and provide a smoother user experience.
