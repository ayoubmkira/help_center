Add new page

1 min read

Expanding your website with additional pages helps organize content and improve navigation. Whether you're creating a new section, a blog post, or a landing page, the website builder makes it easy to add and configure new pages to match your needs.

How to add new page

  1. Open the Pages Manager from the left panel.
  2. Click on the Add New Page button.
  3. A settings popup will appear, allowing you to configure:
    • Page Name: Set a unique name for the page.
    • Page Title: Define the title displayed in the browser tab.
    • Visibility: Choose whether the page is visible in navigation.
    • SEO Settings: Customize meta tags for better search engine optimization.
    • Preloader: Enable or disable a loading animation for the page.
  4. Click Apply to finalize and add the new page.
