Text formatting

There are two ways to format text:

  1. Format a Specific Text Element (affects only one element).
  2. Format All Elements of the Same Type (applies to all similar elements).

Format a Specific Text Element

  1. Click on Text Manager from the left panel.
  2. Hover over the text element (paragraph, header, etc.) that you want to format.
  3. Click the More Options button that appears next to the text.
  4. Apply formatting
    • Bold: Click the bold button to make the text thicker.
    • Italic: Click the italic button to slant the text.
    • Case: Choose between uppercase or lowercase to change the text case.

Format All Elements of the Same Type

  1. Open the Text Manager from the left panel.
  2. All text types (Paragraphs, Headings, Buttons, Navbar Links) are displayed.
  3. Apply formatting using the available options:
    • Bold: Makes all selected text elements thicker.
    • Italic: Slants all selected text elements.
    • Case: Changes all selected text elements to Uppercase or Lowercase.

The changes will apply to all elements of the selected type across the website.
