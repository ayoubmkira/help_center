Changing Font Style & Size

1 min read

Customizing text style is essential for maintaining a consistent design across your website. With the Text Manager, you can easily modify the font, size, and formatting of different text elements, including paragraphs, headers, buttons, and navbar links.

Changing Font style & size

  1. Click on Text Manager in the left bar.
  2. Click on the text you want to edit.
  3. A Panel Control will appear
  4. Change Font Style & Size
    • Font Style: Choose a different font.
    • Size: Adjust text size using the input field or slider.

Note:

Any changes made to the font style, or size will be applied to all elements of the same type across the website. For example, if you adjust the font size for paragraphs, the update will reflect on all paragraph elements site-wide.

Note:

To modify the style or size of an individual text element, you need to use custom CSS code.
