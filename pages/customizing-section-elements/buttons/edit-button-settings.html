Edit button settings

2 min read

With the Edit Button Settings feature, you have full control over the appearance and behavior of buttons on your website. You can change the button type to match your desired style (e.g., primary, success, or danger), adjust its size and width, and add icons for better visual appeal. Whether you want your button to stand out or fit seamlessly into your design, this tool offers the flexibility to make it exactly how you want it.

Edit button settings:

  1. Click on Text Manager or Theme Settings in the left bar.
  2. Hover Over the Desired Button.
  3. Click on the Button Settings Button.
  4. Select the Button Type (e.g., Primary, Success, Danger, Link, etc.).
  5. Choose the Button Size (Large, Defautl, Small, Extra mall).
  6. Set the Button Width to either:
    • Match Parent: expand to fill the container.
    • Wrap Content: fit to content size.
  7. Choose an Icon for the button (None, Left Side, Right Side).
  8. Click on Apply button.
