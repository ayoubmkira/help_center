Change animation

1 min read

Animations can bring your website to life by adding dynamic effects to text, buttons, images, and more. With the animation feature, you can control how and when these elements animate on the page, improving user experience and visual appeal. Adjust the type of animation, timing, duration, and repetition to create seamless transitions and engaging interactions for your users.

Change animation

  1. Click on Theme Settings from the left panel.
  2. Hover over the element (text, button, image, etc.) you want to animate.
  3. Click the Change Animation button.
  4. A settings window will pop up with the following options:
    • Animation Type: Choose the effect (e.g., fade, zoom-in, flip-z).
    • Timing Function: Set how the animation progresses (e.g., ease-in, linear).
    • Duration: Define how long the animation lasts.
    • Delay: Add a wait time before the animation starts.
    • Repeat: Choose how many times the animation plays.
  5. Click the Apply button to save the changes.
