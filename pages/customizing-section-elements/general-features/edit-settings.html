Element settings

2 min read

The Edit Settings feature allows you to fine-tune the appearance and behavior of individual elements on your website. You can adjust spacing, choose different skins, control visibility across devices, and apply custom CSS to tailor the look and feel of each element. Whether you're modifying margins, setting themes, or adding custom styles, this feature offers complete control over your website's design.

Edit element settings

  1. Click on Theme Settings from the left panel.
  2. Hover over the element (text, button, image, etc.) you want to edit.
  3. Click the Element Settings button.
  4. A settings window will pop up with the following options:
    • Margin Top and Bottom: Adjust the top and bottom margins to control spacing.
    • Skin: Choose the skin for the element (Light, Dark, or Default).
    • Visibility: Set the element's visibility for different devices (Phone, Tablet, Desktop).
    • CSS Code: Apply custom CSS styles to the element.
  5. Click the Apply button to save the changes.
