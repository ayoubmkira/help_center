Remove or duplicate an element

1 min read

The Remove and Duplicate options allow you to manage elements on your website with ease. You can quickly delete any element you no longer need or create an identical copy to reuse elsewhere. This flexibility makes it easier to organize your website layout and content without starting from scratch.

Remove or duplicate an element

  1. Click on Theme Settings from the left panel.
  2. Hover over any element that you want to remove or duplicate.
  3. Click the Remove button to delete the element, or Duplicate button to create a copy of the element.
