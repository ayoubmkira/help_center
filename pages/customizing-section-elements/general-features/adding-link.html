Adding link

2 min read

Links are an essential part of navigation and interactivity on your website. With multiple link types available, you can link to external websites, specific sections or pages within your site, images, videos, or even open modals. By following the steps below, you can easily add the right type of link to any text element, enhancing your siteâ€™s functionality and user experience.

  1. Click on Text Manager from the left panel.
  2. Hover over the text element (paragraph, header, button, image, etc) that you want to add a link to.
  3. Click the More Options button that appears next to the element.
  4. Click the Link button that appears next to the element.
  5. Choose Link Type:
    • External Link: Enter the full URL (e.g., https://example.com) to link to an external website.
    • Section/Page Link: Choose the page and section within your website to link to.
    • Image Link: Select an image file to link to, which will open when clicked.
    • Video Popup: Link to a video that will pop up in a modal when clicked.
    • Modal Popup: Link to a modal window that opens when clicked.
  6. Click on Apply button to save.
