Editing form

2 min read

Forms allow users to collect information from visitors. In your builder, you can easily add and customize form fields using a drag-and-drop system and by filling in specific attributes for each input type.

Edit Form Elements

  1. Choose on Theme Settings or Text Manager in the left bar.
  2. Hover over the form.
  3. Drag and drop the desired input field into the form.
  4. Fill in the available attributes based on the selected input type (e.g., placeholder, required, name, value).
  5. Click on Apply button to save changes.
