Edit image settings

2 min read

Use the image utility to easily replace or update images within your sections. This allows you to either select from existing images or upload new ones while ensuring accessibility through alt text.

Replace an Image:

  1. Click on Text Manager or Theme Settings in the left bar.
  2. Hover Over the Desired Image.
  3. Click on the Image Settings Button.
  4. Choose an image or Upload a new one.
  5. Enter Image Alt Text

    Alt text provides a description of the image for accessibility and search engines.

  6. Click on Apply button.
